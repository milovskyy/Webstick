# Настройка автоформатирования

## Проблема: автоформатирование не работает при сохранении

### Быстрое решение

1. **Установите расширение Prettier:**
   - Откройте Extensions (Ctrl+Shift+X)
   - Найдите и установите: `Prettier - Code formatter` (esbenp.prettier-vscode)

2. **Перезагрузите окно редактора:**
   - Ctrl+Shift+P → "Developer: Reload Window"

3. **Проверьте работу:**
   - Откройте любой `.tsx` файл
   - Добавьте лишние пробелы или пустые строки
   - Сохраните файл (Ctrl+S)
   - Код должен автоматически отформатироваться

### Если не работает

#### Вариант 1: Форматирование вручную через команду
```bash
npm run format
```

#### Вариант 2: Форматирование конкретного файла
```bash
npx prettier --write components/Sidebar.tsx
```

#### Вариант 3: Форматирование через палитру команд
1. Откройте файл
2. Нажмите Ctrl+Shift+P
3. Введите "Format Document"
4. Выберите "Prettier - Code formatter"

### Проверка настроек

1. Откройте Command Palette (Ctrl+Shift+P)
2. Введите "Preferences: Open User Settings (JSON)"
3. Убедитесь, что там нет настроек, которые переопределяют настройки проекта

### Проверка вывода Prettier

1. Откройте Command Palette (Ctrl+Shift+P)
2. Введите "Output"
3. В выпадающем списке выберите "Prettier"
4. Сохраните файл и посмотрите, есть ли ошибки

### Установленные зависимости

- ✅ `prettier` - установлен
- ✅ `prettier-plugin-tailwindcss` - установлен (для сортировки классов Tailwind)

### Файлы конфигурации

- ✅ `.prettierrc` - конфигурация Prettier
- ✅ `.vscode/settings.json` - настройки редактора
- ✅ `.prettierignore` - файлы, которые нужно игнорировать
